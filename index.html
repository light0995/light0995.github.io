JavaScript

It It is a programming language used to make websites interactive. It allows me to add dynamic features like animations, form validation, and real-time updates, making the website more engaging and functional

jQuery

jQuery is a JavaScript library that simplifies coding by making it easier to handle tasks like animations, event handling, and manipulating HTML elements. It helps me write less code while achieving more functionality on websites

React

React.js is a JavaScript library used to build fast, interactive user interfaces for web applications. It helps me create reusable components, making development more efficient and improving the performance of the website by updating only the parts that need changes

Node.js is used building server-side applications. It allows me to run JavaScript on the server, handle databases, and manage backend processes, making it possible to create full-stack web applications using one language for both frontend and backend

MongoDB

MongoDB is a NoSQL database used to store and manage data in a flexible, document-oriented format. It allows me to handle large amounts of data efficiently and store it in a more flexible way compared to traditional databases, making it ideal for modern web applications
